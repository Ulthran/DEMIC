## Installation

The development version can be installed from GitHub with:

```r
# install.packages("devtools")
devtools::install_github("Ulthran/DEMIC")
```

## Documentation

This package is intended to be used in the context of the sunbeam extension [sbx_demic](https://github.com/Ulthran/sbx_demic). To use it independently of sunbeam, you'll have to be able to generate the coverage files as input. Please cite Gao, Y., Li, H.
